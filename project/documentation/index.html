Project Overview

Description

Block Burst is a casual puzzle game where players drag and place block pieces onto a grid to complete full rows or columns. The goal is to score as high as possible by clearing lines efficiently while managing limited space. The game ends when no available block can be placed on the board.

Designed for quick play sessions, Block Burst emphasizes clarity, satisfying feedback, and strategic spatial thinking. It is fully playable offline and suitable for players of all ages.

Project Design

Core Gameplay
  • Players are given a set of 3 block pieces each turn.
  • Blocks are placed onto a 8x8 grid via drag-and-drop.
  • Completing full rows or columns clears it from the board.
  • Cleared lines award points.
  • The game continues endlessly until no valid placement remains.
Scoring System
  • Points are awarded for each cleared row or column.
  • Multiple line clears in a single move grant bonus points.
  • Optional combo or streak multipliers added for advanced play.
  • Perfect placing earns more bonus points.
  • Clear the whole board get a huge point amout and change the game theme.
Logic Flow
1 Start with an empty grid or random pattern
2 Receive a set of 3 block pieces from algorithm
3 Place blocks strategically
4 Clear lines to create space and earn points
5 Repeat until no moves are possible
6 Display final score and restart option
Difficulty & Progression
  • Difficulty increases naturally as grid space becomes limited.
  • Larger and more complex block shapes increase challenge over time.
  • No timers or pressure mechanics—players control the space.
Visual Design
  • Clean, colorful block visuals for strong readability
  • Diverse themes set from editor
  • Clear animations when placing blocks and clearing lines
  • Satisfying effects to reinforce successful moves
